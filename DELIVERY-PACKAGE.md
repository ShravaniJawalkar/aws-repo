### app-enhanced.js
Complete Node.js/Express application with:
- **Subscription Endpoints** (3):
  - POST `/api/subscribe` - Email subscription
  - POST `/api/unsubscribe` - Email unsubscription
  - GET `/api/subscriptions` - List subscriptions

- **Upload Endpoints** (2):
  - POST `/api/upload` - Upload and queue notification
  - GET `/api/download/:eventId/:fileName` - Download placeholder

- **Admin Endpoints** (4):
  - GET `/health` - Health check
  - GET `/admin/queue-status` - Queue metrics
  - POST `/admin/process-queue` - Manual queue processing
  - POST `/admin/send-test-message` - Test message

- **Features**:
  - SQS message publishing
  - SNS email subscriptions
  - Background worker (30-second polling)

  - Logging
